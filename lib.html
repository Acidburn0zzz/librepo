Handle options¶

LRO (aka LibRepo Option) constants are used to set Handle options via setopt() method.

This options could be also set by Handle class methods with same names in lowercase and without LRO_ prefix.

Example:

# The command:
h.setopt(librepo.LRO_URL, "http://ftp.linux.ncsu.edu/pub/fedora/linux/releases/17/Everything/i386/os/")
# is equivalent to:
h.url("http://ftp.linux.ncsu.edu/pub/fedora/linux/releases/17/Everything/i386/os/")

Note

For detailed description of this options consult Handle page.

Error codes¶

LibRepo Error codes.

librepo.LRE_OK¶

Everything is ok.

librepo.LRE_BADFUNCARG¶

Bad function argument(s).

librepo.LRE_BADOPTARG¶

Bad argument for the option in setopt().

librepo.LRE_UNKNOWNOPT¶

Unknown option.

librepo.LRE_CURLSETOPT¶

cURL doesn’t know an option used by librepo. Probably too old cURL version is used.

librepo.LRE_ALREDYUSEDRESULT¶

Result object is not “clean” (it has been already used and filled).

librepo.LRE_INCOMPLETERESULT¶

Result object doesn’t contain all what is needed.

librepo.LRE_CURLDUP¶

Cannot duplicate cURL handle. No free memory?

librepo.LRE_CURL¶

A cURL error.

librepo.LRE_CURLM¶

A multi cURL handle error.

librepo.LRE_BADSTATUS¶

An error HTTP or FTP status code.

librepo.LRE_TEMPORARYERR¶

An error that should be temporary (e.g. HTTP status codes 500, 502-504, ..)

librepo.LRE_NOTLOCAL¶

If LRO_UPDATE option is True and URL is not a local path (address).

librepo.LRE_CANNOTCREATEDIR¶

Cannot create directory for downloaded data. Bad permission?

librepo.LRE_IO¶

Input/Output error. Bad permission?

librepo.LRE_MLBAD¶

Bad mirrorlist or metalink file. E.g. metalink doesn’t contain reference to target file (repomd.xml), mirrorlist is empty, ..

librepo.LRE_MLXML¶

Cannot parse metalink xml. Non-valid metalink file?

librepo.LRE_BADCHECKSUM¶

A downloaded file has bad checksum.

librepo.LRE_REPOMDXML¶

Cannot parse repomd.xml file. Non-valid repomd.xml file?

librepo.LRE_NOURL¶

No usable URL found. E.g. bad links or no links in metalink.

librepo.LRE_CANNOTCREATETMP¶

Cannot create temporary directory.

librepo.LRE_UNKNOWNCHECKSUM¶

Unknown type of checksum is needed for verification one or more files.

librepo.LRE_BADURL¶

Bad URL specified.

librepo.LRE_UNKNOWNERROR¶

An unknown error.
